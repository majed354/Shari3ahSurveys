# Shari3ahSurveys

موقع عربي ثابت بعنوان: `ساحة الاستطلاعات والآراء لبرامج كلية الشريعة`.

الفكرة:
- تنظيم الاستطلاعات بما يخدم الدراسة الذاتية والاعتماد البرامجي.
- التبويب بحسب محاور الاستفادة في الدراسة الذاتية، لا بحسب الجهة فقط.
- دعم فلترة البرنامج والسنة من الواجهة.

الملفات:
- `index.html`
- `css/style.css`
- `js/app.js`
- `js/surveys-data.js`
- `scripts/build_survey_data.py`
- `استطلاعات_وتقييمات_كلية_الشريعة_1445_1446.xlsx`

التشغيل:
- افتح `index.html` مباشرة.
- أو اربطه لاحقًا مع GitHub وNetlify بوصفه موقعًا static.

تحديث البيانات:
- ملف Excel الحالي هو المصدر الرسمي للبيانات.
- لتجديد بيانات الموقع بعد تحديث الملف شغّل:

```bash
python3 scripts/build_survey_data.py
```

- هذا الأمر يعيد توليد الملف `js/surveys-data.js` تلقائيًا من ملف Excel.
